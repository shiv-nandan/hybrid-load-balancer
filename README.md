# Hybrid Load Balancer

## Overview

Hybrid Load Balancer is a project that implements an intelligent load balancing system designed to efficiently distribute incoming tasks across multiple servers. The system combines multiple load balancing strategies to improve performance, reduce response time, and optimize resource utilization.

This project demonstrates practical implementation of algorithm design, system simulation, and performance evaluation.

---

## Objectives

- Distribute workload efficiently across servers
- Minimize response time
- Prevent server overload
- Improve overall system performance
- Compare different load balancing strategies

---

## Features

- Simulation of multiple servers
- Task distribution using hybrid strategy
- Combination of static and dynamic load balancing approaches
- Performance metric calculation
- Visualization of server load (if graphs included)
- Scalable and modular design

---

## Load Balancing Strategies Used

- Round Robin
- Least Connections
- Weighted Distribution
- Hybrid Decision Logic (custom combination of strategies)

## How It Works

1. Multiple virtual servers are initialized.
2. Incoming tasks are generated or simulated.
3. The hybrid algorithm selects the best server based on predefined logic.
4. Server loads are updated dynamically.
5. Performance metrics are calculated and analyzed.
